# 🌌 Stellar Object Classification & Analysis System

An AI-powered system to classify celestial objects such as **Stars ⭐, Galaxies 🌌, and Quasars ⚡** using Machine Learning and Deep Learning.

---

## 🚀 Project Overview

With the rapid growth of astronomical data from surveys like SDSS, manual classification of celestial objects has become inefficient.

This project builds a **hybrid AI system** that:
- Classifies stellar objects using **tabular data**
- Supports **image-based classification (CNN - future scope)**
- Provides **explainable predictions**
- Offers an **interactive web interface**

---

## ✨ Features

✅ Stellar Object Classification (Star / Galaxy / Quasar)  
✅ Machine Learning Models:
- Random Forest  
- XGBoost  
- Decision Tree  
- Logistic Regression  

✅ Explainable AI (SHAP)  
✅ Interactive Web App (Streamlit)  
✅ CSV Upload for Bulk Prediction  
✅ Knowledge Engine (Educational Insights)  
✅ (Upcoming 🚀) Image Classification using CNN  
✅ (Upcoming 🚀) NASA API Integration  
✅ (Upcoming 🚀) GPT-based Explanations
